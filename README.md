# Bikersenzameta

Sito vetrina per negozio di abbigliamento e attrezzatura moto: mondo strada e mondo offroad/MTB.

**Live**: https://trikos.github.io/bikersenzameta/

## Stack

- Vite
- React 18
- Tailwind CSS 3

## Sviluppo

```bash
npm install
npm run dev       # dev server
npm run build     # build produzione
npm run preview   # preview build locale
```

## Deploy

Deploy automatico su GitHub Pages tramite GitHub Actions a ogni push su `main`.
